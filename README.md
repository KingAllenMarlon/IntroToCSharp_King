# ITELEC2 Prelim Activity 01: Codac Logistics Delivery & Fuel Auditor

Name: Allen Marlon King

Task Description:
You have been hired by Codac Logistics to develop a console-based tool for their delivery drivers. The company needs to track the daily fuel expenses and delivery performance of a single vehicle over a 5-day work week.
Your task is to build a tool that gathers driver profiles, validates distance data, calculates fuel efficiency ratings, and provides a financial summary to the accounting department.
